# Interactive Player Performance Monitoring Dashboard

This project is a polished prototype for exploring how interactive dashboards can support serious-game design, player learning analysis, and behavior monitoring.

## What it does
- Generates synthetic player session data for five sample players
- Visualizes score progression across sessions
- Compares players by score, engagement, completion, errors, and learning gain
- Provides a clean, presentation-ready interface for portfolio storytelling

## Why this project stands out
- Demonstrates strong Python data handling with pandas
- Uses Streamlit to create an interactive analytics experience
- Separates reusable analytics logic from app presentation for maintainability
- Includes a simple testing setup to support future iteration

## Tech stack
- Python
- pandas
- Streamlit
- pytest

## How to run
```bash
python -m streamlit run app.py
```

## Project purpose
This is intended as a portfolio-ready prototype for learning, experimentation, and showcasing practical data storytelling in a real-world analytics context.
