# Interactive Player Performance Monitoring Dashboard

This project is a small example prototype for a serious-games research idea: monitoring player performance, engagement, and learning behavior through an interactive dashboard.

## What it does
- Generates synthetic player session data
- Visualizes player progress over time
- Compares players by score, engagement, completion, and learning gain
- Presents a clear example of how monitoring could support serious-game design and learning analysis

## Tech stack
- Python
- pandas
- Streamlit

## How to run
```bash
python -m streamlit run app.py
```

## Project purpose
This is intended as an early-stage prototype for learning, experimentation, and portfolio presentation.
